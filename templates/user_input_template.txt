How To Use This File:
Run these commands from inside the project folder.
Each workflow run will be saved in its own folder inside `projects/`.
Step 1: Replace the example text below with your own answers in simple English.
Step 2: Save this file.
Step 3: Start the workflow with:
PYTHONPATH=src python3 -m literature_review_agent --project-root . start-from-file --input-file templates/user_input_template.txt
Optional: if you want to choose the project folder name yourself, add:
--project-name my_project_name
Step 4: Run the workflow:
PYTHONPATH=src python3 -m literature_review_agent --project-root . run
Step 5: If the workflow pauses, review the shortlist:
PYTHONPATH=src python3 -m literature_review_agent --project-root . show-shortlist
Step 6: Approve the shortlist:
PYTHONPATH=src python3 -m literature_review_agent --project-root . approve-shortlist --all
Step 7: Run the workflow again:
PYTHONPATH=src python3 -m literature_review_agent --project-root . run
Step 8: Read the final result:
PYTHONPATH=src python3 -m literature_review_agent --project-root . show-final-brief
The output will be saved inside:
projects/<project_name>/
